File Format Requirements
  • Supported Files: CSV (.csv), Excel (.xlsx, .xls)
  • Identifier Column: SKU, Part Number, Customer Part Number, Master Item Number, or UPC
  • Price Column: Unit Price, Price, Cost, or similar
  • Headers: First row must contain column headers
  • Price Format: Numbers only ($ and commas will be removed automatically)
How It Works
  1. Upload CSV or Excel file with part identifiers and new prices
  2. System automatically matches parts by:
    • SKU / Part Number
    • Customer Part Number
    • Master Item Number
    • UPC Code
  3. Old prices are archived to history
  4. New prices become current
  5. Full audit trail is maintained
Note: Price changes are immediate and affect all future calculations.
